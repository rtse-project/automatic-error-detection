# JAVA-XAL 
![Build](https://rtse-isys.aau.at/giovanni.liva/java-xal/badges/master/build.svg)

In order to package the project first call 
```bash
mvn compile
```

Then call 
```bash
mvn package
```

This project uses the ANTLRv4 parser to convert Java source code to XAL files.
